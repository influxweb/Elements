JavaScript

In an effort to have Elements be as lightweight as possible, it does not ship with jQuery. Instead, we have chosen to use a jQuery alternative called Cash. If Cash is not robust enough for a site built on Elements, it can be replaced with jQuery with no loss of functionality in the core. The majority of JavaScript used in Elements is "vanilla" in nature. We have found this to perform well and work cross-browser.

While you can create and call JavaScript function using the familiar $('#some_ID') or $('.some_CLASS') way, the preferred way in Elements is to use a data attribute called hook, $.hook('some_data_hook'). Using data-hook allows for having multiple possible elements, like when using a class.
